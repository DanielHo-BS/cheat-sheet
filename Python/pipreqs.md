# pipreqs

## Introduction

Pip requirements.txt generator based on imports in project

## Installation

```bash
pip install pipreqs
```

## Usage

```bash
cd <project>
# pipreqs [options] [<path>]
pipreqs <path>
# Successfully saved requirements file in <path>/requirements.txt
```
